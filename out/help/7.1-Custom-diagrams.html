7.1 Custom diagrams

All predesigned reports were built only using settings that can be modified in User Inteface. As a result you can build your own entirely unique charts like How many different artists were drawing MTG cards every year.

Customizing report is conceptually similar to creating Pivot Table / Pivot Chart in Microsoft Excel or Libre Office Calc. In case you are not familiar with Pivot tables there is a Wikipedia page on the subject.

To create a custom graph

  • select graph data source: Deck, Collection, All cards
    • use Filter by search result checkbox to apply the filters from main window. This way you can use all the filters like Search text, Filter buttons, Filter by Legality and so on to confine the data used when building a graph.
  • setup Argument fields - correspond to X axis
  • setup Series fields - correspond to different stacks in stacked bars.
  • setup Aggregate fields - aggreate field is used to produce Y value
    • select aggregate type: sum, count, min, max, average by clicking aggregate field caption
  • setup sorting by argument / series / aggregate fields by clicking field captions
  • select the chart type: Stacked column, Spline, Circular and so on
  • click Build!
