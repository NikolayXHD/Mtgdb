Prerequisites

The program requires .NET Framework v4.0 Most people have it already installed. In case you don't download it from https://www.microsoft.com/en-gb/download/details.aspx?id=17851

.NET Framework v4.0 itself can run on Windows XP or newer versions of MS Windows

Running Mtgdb.Gui

Run Mtgdb.Gui shortcut in the root of extracted .zip

Downloading Images

Press the red cloud button in top right corner of Mtgdb.Gui window. An update window will open where you will see buttons to download images.
